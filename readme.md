# 1 全局安装 mock server

- 安装

  `https://github.com/typicode/json-server`

- 安装成功后，执行下面命令确认

  `npm list --depth=0 -g`



# 2 下载代码

  `https://github.com/jiajianrong/easy-react`

- 安装依赖

  `npm install --save`



# 3 启动mock server

- 进入目录 fake_server， 执行

  `json-server --watch cfqbms.json --routes cfqbms-routes.json`



# 4 启动项目

- 运行

  `npm run start`



# 5 编译给后端

  `npm run build`
